## ✨ Features

### 🎯 Core Gameplay
- **Arrow key controls** with smooth input handling
- **Progressive difficulty** - starts slow, gets faster as you score
- **Dynamic scoring system** with level progression
- **Wall and self-collision detection**
- **High score persistence** using localStorage

### 🎨 Visual Effects
- **Particle explosion system** for food collection and collisions
- **Pulsing food animation** with neon glow
- **Gradient snake body** with glowing effects
- **Eye animations** that follow movement direction
- **Glass morphism UI** with animated backgrounds
- **Smooth transitions** and hover effects

### 🎵 Audio
- **Web Audio API** sound effects for:
  - Food collection (pleasant chime)
  - Game over (low tone)
  - Pause/resume (feedback tones)

### 📊 Game Statistics
- Real-time **score tracking**
- **High score** saved locally
- **Level system** (increases every 5 points)
- **Speed indicator** showing current difficulty

## 🚀 How to Play

1. Click **"START GAME"** or press **R**
2. Use **Arrow Keys** (↑ ↓ ← →) to control the snake
3. Eat the **red food** to grow and increase your score
4. Avoid hitting the **walls** or your **own tail**
5. Game speed **increases gradually** as your score grows

### ⌨️ Keyboard Shortcuts
- **Arrow Keys** - Move snake
- **P** - Pause/Resume game
- **R** - Restart game

## 🛠️ Technologies Used

- **HTML5 Canvas** - Game rendering
- **CSS3** - Animations, gradients, glass morphism
- **Vanilla JavaScript** - Game logic, particle system
- **Web Audio API** - Sound effects
- **LocalStorage API** - High score persistence
