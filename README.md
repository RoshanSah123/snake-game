## ✨ Features

### 🎮 Core Gameplay
- **Smooth Controls** - Arrow key navigation with input queuing
- **Progressive Difficulty** - Speed increases gradually from slow to fast
- **Level System** - New level every 5 points
- **High Score Tracking** - Persistent storage using localStorage
- **Pause/Resume** - Press 'P' or click button

### 🎨 Visual Effects
- **Particle System** - Explosions when eating food and collisions
- **Neon Glow Effects** - Glowing snake and food elements
- **Pulsing Food** - Animated food with heartbeat effect
- **Gradient Snake** - Smooth color transitions
- **Snake Eyes** - Directional eyes on snake head
- **Glass Morphism** - Modern translucent UI elements
- **Animated Background** - Dynamic grid pattern

### 🔊 Audio
- **Eating Sound** - Satisfying tone when collecting food
- **Game Over Sound** - Dramatic ending notification
- **Menu Sounds** - Button click feedback

### 📊 UI Features
- **Real-time Stats** - Score, High Score, Level display
- **Speed Indicator** - Current speed percentage
- **Start Screen** - Game instructions and preview
- **Pause Overlay** - Clean pause screen
- **Game Over Modal** - Score summary and options

## 🚀 How to Play

1. **Start** - Click "START GAME" or press 'R'
2. **Control** - Use arrow keys (↑ ↓ ← →) to move the snake
3. **Goal** - Eat red food pellets to grow and increase score
4. **Avoid** - Hitting walls or your own tail
5. **Speed** - Game gets progressively faster as score increases
6. **Pause** - Press 'P' or click PAUSE button
7. **Restart** - Press 'R' or click RESTART button
